## Milestone Summary

**Key Decisions:**

- Host Vercel; Postgres Neon; images Cloudflare R2 — all free tier, no self-hosted dependency
- Drop Redis entirely; better-auth sessions on Postgres
- Delete the job-search pipeline outright (not migrate); delete homelab-monitoring admin entirely (`/admin` 404s) — FUTURE-02 tracks re-enable
- Recipes are file-based MDX, text-only, public — no DB, no login
- Pin eslint 9.39.4 (10.x incompatible with eslint-plugin-react 7.x) + kysely 0.28.17 via resolutions (better-auth 1.6.x adapter requires 0.28.x) to satisfy the Aikido Safe Chain min-age gate
- Next.js 16 `proxy.ts` (renamed from middleware) at `src/proxy.ts` with `export function proxy` — CSP scoped to `/admin/*`, kept dormant for FUTURE-02

**Issues Resolved:**

- K3s homelab offline (flood) → fully re-platformed onto Vercel + Neon + Cloudflare R2 with valid TLS
- `bun install` blocked by Aikido Safe Chain age-gate → permanent fix via aged-pins + Renovate security-only
- pg SSL deprecation warning → `withStrictSsl()` rewrites `sslmode` → `verify-full` in `src/lib/prisma.ts`
- better-auth 403 on social sign-in → `trustedOrigins` (www vs apex)
- Owner role stuck as "member" → OWNER_EMAIL env sourcing + idempotent self-heal databaseHook
- Cloudflare 1033 (dead tunnel) + amazon-only CAA → Vercel DNS records + letsencrypt.org CAA
- Broken homepage thumbnails in prod → read-side components switched from raw R2 keys to the `/api/images` proxy

**Post-100% enhancements (shipped on top of the milestone scope):**

- Global light/dark theme system (next-themes): Ivory & Terracotta light (default) / navy dark, adaptive nav + footer + user toggle. Temp `/theme-preview` mockup page removed.
- Menu print emits full cookable recipes (ingredients + instructions) per item, not just names.

**Technical Debt / Deferred:**

- FUTURE-01: restore homelab Postgres → migrate data into Neon when equipment returns
- FUTURE-02: re-enable live homelab monitoring (un-park the CLOUD-04 dashboard)
- FUTURE-03: remaining recipe back-matter (Menu Making prose, ~100-menu Menus section, Table Service)
- FUTURE-04: recipe full-text search across ingredients/steps (RECIPE-01 is name-search only)
- One restored seed photo (`d9c2e950…`) renders the SVG placeholder — its original file was never migrated off the offline NAS to R2 (graceful degradation by design; pairs with FUTURE-01)
