    **Step A — Pick a job with a tailored resume and a populated pdf_data.**

    Query the n8n DB (or ask Plan 21-01's output for the candidate `job_id`):
    ```sql
    SELECT j.id, j.title, j.company, LENGTH(tr.pdf_data) AS pdf_size
    FROM jobs j
    JOIN tailored_resumes tr ON tr.job_id = j.id
    WHERE tr.pdf_data IS NOT NULL
    ORDER BY tr.generated_at DESC
    LIMIT 3;
    ```
    Note the job id for use below (e.g., `<id>`).

    **Step B — Verify Download PDF.**

    1. Open https://thehudsonfam.com/admin/jobs (or http://localhost:3000 in production mode — `npm run build && npm run start`)
    2. Sign in as the owner
    3. Find the row for `<id>` (search or scroll to it)
    4. Click the row → detail sheet opens
    5. Locate the Tailored Resume section (between Cover Letter and Company Intel)
    6. Confirm the meta row shows `[FreshnessBadge] [Copy icon button] [Download PDF link]`
    7. Click `Download PDF`
    8. Expected: browser download prompt for a file named `tailored-resume-job-<id>.pdf`; saving and opening it in the OS default PDF viewer yields a readable PDF matching the owner's tailored resume content

    Fallback CLI verification:
    ```bash
    curl -sI -b "better-auth.session=<owner-session-cookie>" \
      https://thehudsonfam.com/api/jobs/<id>/tailored-resume-pdf | \
      grep -E 'HTTP/|Content-Type|Content-Disposition|Content-Length'
    # Expect: HTTP/1.1 200 OK
    #         Content-Type: application/pdf
    #         Content-Disposition: attachment; filename="tailored-resume-job-<id>.pdf"
    #         Content-Length: <number > 1000>
    ```

    **Step C — Verify Copy.**

    1. In the same detail sheet, click the ghost Copy icon button between the FreshnessBadge and Download link
    2. Expected (a): sonner toast appears at the bottom-right corner with "Resume copied to clipboard"
    3. Expected (b): the Copy icon morphs to a green / accent Check icon for ~2 seconds, then reverts
    4. Paste into a plain-text editor (e.g., gedit, VS Code, TextEdit in plain mode) — verify the pasted content is the raw markdown with `#` / `**` / list markers (NOT plain-text flattened)
    5. Paste into a rich-text target (e.g., Gmail compose window or a Google Doc) — verify the markdown auto-renders into formatted text (confirms AI-ACTION-01 success)

    **Step D — Confirm empty-state copy for a job WITHOUT artifacts.**

    Pick a job id that has NO cover letter, NO tailored resume, AND NO company research (common for unranked jobs):
    ```sql
    SELECT j.id, j.title
    FROM jobs j
    LEFT JOIN cover_letters cl ON cl.job_id = j.id
    LEFT JOIN tailored_resumes tr ON tr.job_id = j.id
    LEFT JOIN company_research cr ON cr.id = j.company_research_id
    WHERE cl.id IS NULL AND tr.id IS NULL AND cr.id IS NULL
    LIMIT 1;
    ```

    Open the detail sheet for that job. Expected:
    - Cover Letter heading + `No cover letter yet.` italic body
    - Tailored Resume heading + `No tailored resume yet.` italic body
    - Company Intel heading + `No company research yet.` italic body
    - NO FreshnessBadge / Copy button / Download link / Quality badge in any section
